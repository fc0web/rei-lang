## 概要

別数理構築理論（AMRT）は、Category C の最後の理論として、
「同じ問題に対して根本的に異なる解法が必ず存在する」ことを公理化し、
全25公理の超越的統合を達成する。

C1〜C5が「内側」、U1〜U5が「外側」、N1〜N5が「横」、M1〜M5が「下」を
公理化したのに対し、A1〜A5は「斜め」— 全方向を横断する視点を提供する。

---

## 5公理の構造

```
A1（別解存在）   : 別の解き方が必ず存在する       — 多門
A2（非還元性）   : 別解は別解として独立している   — 不二
A3（相補性）     : 別解同士は補い合う             — 相即
A4（選択原理）   : 最適な別解を選べる             — 択法
A5（超越的統合） : 全てを超えた統合がある         — 円融
```

### 別解の理解の過程

```
A1 → A2 → A3 → A4 → A5
存在   独立性  補完   選択   統合

「別解がある」→「本当に別」→「補い合う」→「選べる」→「超えて統合する」
```
