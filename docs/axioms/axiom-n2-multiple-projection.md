## 公理の定義

**公理N2（多重射影公理）：**

同一の対象 O に対して、複数の異なる射影 π₁, π₂, ... が存在し得る。
各射影は O の異なる側面を捉え、それぞれが構造保存的（N1を満たす）である。

```
∀O ∈ World, ∃{π₁, π₂, ..., πₖ} : k ≥ 1
  where ∀i : πᵢ satisfies N1
  and   ∃i,j : πᵢ(O) ≠ πⱼ(O)
```

### 形式的定義

```
定義 N2.1（射影族 Π(O)）：
  対象 O に対する全ての構造保存的射影の集合を Π(O) と書く。
  
  Π(O) = { π | π : O → 𝕄-Space, π satisfies N1 }
  
  N1 より |Π(O)| ≥ 1 が保証されている。
  N2 は |Π(O)| ≥ 2 となる場合の理論。

定義 N2.2（射影の視点 Viewpoint）：
  射影 π は「何を中心に据えるか」という視点 v に対応する。
  
  v : O → center(π(O))
  
  異なる視点は異なる射影を生む：
  v₁ ≠ v₂ ⟹ π_{v₁}(O) ≠ π_{v₂}(O)
  
定義 N2.3（視点空間 V(O)）：
  対象 O に対する全ての可能な視点の集合を V(O) と書く。
  
  V(O) と Π(O) は自然な全射対応を持つ：
  viewpoint_map : V(O) ↠ Π(O)
```

## 仏教哲学との対応

**一即多・多即一（ekam eva anekam）**

一つの対象が多くの射影を持ち（一即多）、
多くの射影が一つの対象を指す（多即一）。

華厳経の「因陀羅網（インドラの網）」のように、
一つの宝珠が全ての宝珠を映し、全ての宝珠が一つの宝珠を映す。
各射影は「一つの宝珠から見た景色」である。

```
一つの音楽（一）
  ├→ スペクトル射影（多の一つ）
  ├→ リズム射影（多の一つ）
  ├→ 和声射影（多の一つ）
  └→ 感情射影（多の一つ）

各射影はそれぞれ完全な 𝕄 であり（一即多）、
全てが同じ音楽を指している（多即一）。
```

**空（śūnyatā）の深化：**

N1で「対象は 𝕄 に射影できる」と述べた（色即是空）。
N2は「その射影は一つに定まらない」と述べる。
これは対象に固定的な「本質」がないこと — まさに空の意味 — を示す。
対象の「本当の姿」は存在しない。あるのは視点に応じた無数の射影のみ。

## 構文への影響

### 多重射影の列挙

```rei
// 音楽に対する複数の射影
music = Sound.load("symphony_no5.wav")

// スペクトル射影
music |> project(:spectral)
// → 𝕄{ 440.0; 220.0, 660.0, 880.0, ... }

// リズム射影
music |> project(:rhythmic)
// → 𝕄{ :beat_4_4; :quarter, :quarter, :quarter, :quarter }

// 和声射影
music |> project(:harmonic)
// → 𝕄{ :c_minor; :i, :iv, :V, :i }

// 感情射影（σ.will を用いた傾向性による射影）
music |> project(:emotional)
// → 𝕄{ :tension; :anticipation, :resolution, :climax }
```

### 視点の切り替え `view_as`

```rei
// 同一対象を異なる視点で操作
triangle = Shape.new(vertices: [(0,0), (1,0), (0.5, 0.87)])

// 頂点視点：各頂点を操作
triangle |> view_as(:vertices) |> map(|v| v |> scale(2.0))

// 辺視点：各辺を操作
triangle |> view_as(:edges) |> map(|e| e |> subdivide(3))

// 面積視点：面積に基づく操作
triangle |> view_as(:area) |> transform(|a| a * 4)
```

### 射影族の一括取得 `projections`

```rei
// 対象の全利用可能射影を列挙
music |> projections
// → [:spectral, :rhythmic, :harmonic, :emotional, :waveform, ...]

// 射影族全体に対する操作
music |> projections |> map(|p| music |> project(p) |> center)
// → [440.0, :beat_4_4, :c_minor, :tension, ...]
// 各射影の「中心」だけを集めた — 対象の多面的な要約
```

### カスタム射影の定義

```rei
// ユーザー定義の射影
projector :color_temperature = Projector.new(
  source: Image,
  center_fn: |img| img |> average_color |> to_kelvin,
  periphery_fn: |img| img |> regions |> map(|r| r |> average_color |> to_kelvin),
  preserves: [:color_distinction, :spatial_adjacency]
)

photo |> project(:color_temperature)
// → 𝕄{ 5500K; 3200K, 6500K, 4800K, 7100K }
```

## σ（6属性）との関係

```
N2 は σ の各属性に対して「その属性を中心に据えた射影」を自然に定義する：

π_field(O)    : 𝕄{ O.σ.field;    ... }   場を中心に据えた射影
π_flow(O)     : 𝕄{ O.σ.flow;     ... }   流れを中心に据えた射影
π_memory(O)   : 𝕄{ O.σ.memory;   ... }   記憶を中心に据えた射影
π_layer(O)    : 𝕄{ O.σ.layer;    ... }   層を中心に据えた射影
π_relation(O) : 𝕄{ O.σ.relation; ... }   関係を中心に据えた射影
π_will(O)     : 𝕄{ O.σ.will;     ... }   意志を中心に据えた射影

すなわち、6属性は「6つの基本視点」を提供する。
任意の対象は少なくとも6通りの 𝕄 射影を持つ。
```

## 数学的定理

```
定理 N2.1（射影族の豊富性定理）：
  |Obs(O)| = n なる対象 O に対して、
  |Π(O)| ≥ n である。
  
  すなわち、対象の弁別可能な特徴の数だけ、
  少なくともそれだけの射影が存在する。
  
  証明の骨子：
  各観測関数 obsᵢ を中心に据えた射影 πᵢ が構成可能。

定理 N2.2（6属性射影の基底性定理）：
  σ の6属性に対応する6射影 {π_field, π_flow, π_memory, 
  π_layer, π_relation, π_will} は、
  射影族 Π(O) の「基底」を成す。
  
  すなわち、任意の射影 π ∈ Π(O) は、
  この6基本射影の重み付き合成として表現できる：
  
  π = α₁·π_field + α₂·π_flow + ... + α₆·π_will
  
  ここで + は射影の合成操作（N4で厳密化）。

定理 N2.3（射影の順序定理）：
  射影族 Π(O) には自然な半順序が存在する。
  
  π₁ ≤ π₂ ⟺ π₁ が保存する観測関数集合 ⊆ π₂ が保存する観測関数集合
  
  すなわち「より多くの特徴を捉える射影」が上位。
  最大元は N5 の不完全性公理により存在しない。

定理 N2.4（C2傾向性との接続定理）：
  対象 O の傾向性 τ(O) は、射影族 Π(O) 上の確率分布として解釈できる。
  
  τ(O) → P(π) : 「O が自然に取りやすい射影」の分布
  
  すなわち、意志を持つ対象は「見られたい角度」を持つ。
  これは C2（傾向性公理）の N2 における表現である。

定理 N2.5（視点の等価性定理）：
  いかなる射影も他の射影に対して「特権的」ではない。
  
  ∀πᵢ, πⱼ ∈ Π(O) : rank(πᵢ) と rank(πⱼ) は比較不能であり得る
  
  これは民主的認識論 — 全ての視点は等しく正当 — の形式化である。
  ただし、特定の問題に対してはある射影が他より「有用」である（実用的優先性）。
```

## C1〜C5・U1〜U5との接続

```
C1（自己参照）→ N2：値が σ(v) で自己参照するとき、
                    「どの射影で自分を見るか」の選択が生じる。
                    自己認識の多面性。

C2（傾向性）  → N2：傾向性 τ は「好まれる射影」の方向を示す。
                    ある音楽が「和声的に聴かれたがる」こと。

C5（共鳴）    → N2：異なる射影で同じ構造が現れたとき共鳴が生じる。
                    「和声射影の音楽」と「色彩射影の絵画」が同じ 𝕄 構造
                    → 共感覚の数学的基盤。

U1（構造還元）→ N2：U1 の還元が一意でないことの証明。
                    同じ対象が複数の 𝕄 に還元される。

U4（領域架橋）→ N2：異なる領域の対象が同じ射影構造を共有するとき、
                    U4 の架橋が N2 を通じて実現される。
```

*N3（射影間変換公理）への布石：*

N2は「複数の射影がある」と宣言した。しかし、射影同士は孤立しているのか？
N3は「射影と射影の間を行き来できる」と宣言する。フーリエ変換が時間領域と
周波数領域を行き来するように、あらゆる射影間に「翻訳」が存在する。
これはReiのパイプ `|>` が「射影間の橋渡し」として機能する理論的根拠を与える。
