## 公理の定義

**公理N1（射影存在公理）：**

任意の対象 O（図形・音・画像・テキスト・その他いかなる認識可能な存在）に対して、
その本質的構造を保存する 𝕄 への射影 π が少なくとも1つ存在する。

```
∀O ∈ World, ∃π : O → 𝕄{c; p₁, p₂, ...}
  such that Distinguish(O, x, y) ⟹ Distinguish(π(O), π(x), π(y))
```

ここで `Distinguish(S, a, b)` は「構造 S において a と b が弁別可能である」ことを意味する。

### 形式的定義

```
定義 N1.1（対象空間 World）：
  World = 認識可能な全ての存在の集合
  = Numbers ∪ Shapes ∪ Sounds ∪ Images ∪ Texts ∪ Relations ∪ ...
  
  Numbers ⊂ World       （数は対象の特殊ケース）
  
定義 N1.2（射影 π）：
  π : World → 𝕄-Space
  
  ここで 𝕄-Space は全ての中心-周囲パターンの空間。
  
  π は以下を満たす：
  (a) 構造保存性： O の弁別可能な特徴が π(O) で保存される
  (b) 有限性：     π(O) の周囲要素数は有限である
  (c) 計算可能性： π の計算は有限時間で完了する
  
定義 N1.3（構造保存性の厳密化）：
  対象 O に対して、観測関数の集合 Obs(O) = {obs₁, obs₂, ...} が存在する。
  射影 π が構造保存的であるとは：
  
  ∀obs ∈ Obs(O), ∃obs' ∈ Obs(π(O)) :
    obs(a) ≠ obs(b) ⟹ obs'(π(a)) ≠ obs'(π(b))
  
  すなわち、元の対象で区別できるものは、射影後も区別できる。
```

## 仏教哲学との対応

**色即是空（rūpaṃ śūnyatā）**

「色（形あるもの）は空（構造的関係性）である」— 般若心経の核心命題。

N1はこれを数学的に述べている：形あるもの（図形・音・画像）は全て構造（𝕄）に射影できる。
対象の「実体」は中心-周囲の関係性パターンにある。物そのものの中に本質があるのではなく、
構造的関係の中に本質がある。

```
色（rūpa）    = 対象 O          ← 認識される形あるもの
空（śūnyatā）  = 𝕄{c; p₁,...}    ← 構造的関係性

色即是空 : O は 𝕄 に射影可能
空即是色 : 𝕄 から O を再構成可能（N3で補完される）
```

**五蘊（skandha）との対応：**

```
色（rūpa）     : 物質的対象 → GFT（幾何形状）の射影

想（saṃjñā）   : 概念的対象 → テキスト・記号の射影
行（saṃskāra） : 意志的対象 → C1〜C5（意識数理学）の射影
識（vijñāna）  : 認識そのもの → 射影 π 自体が識の作用
```

## 構文への影響

### 射影構文 `project`

```rei
// 図形の射影
triangle = Shape.new(vertices: [(0,0), (1,0), (0.5, 0.87)])

triangle |> project(:vertex_center)
// → 𝕄{ (0.5, 0.29);                  // 中心 = 重心
//       (0,0), (1,0), (0.5, 0.87)      // 周囲 = 頂点
//    }

triangle |> project(:edge_center)
// → 𝕄{ triangle;                      // 中心 = 図形全体
//       edge₁, edge₂, edge₃            // 周囲 = 辺
//    }
```

```rei
// 音の射影
chord = Sound.new(frequencies: [261.6, 329.6, 392.0])  // Cメジャー

chord |> project(:spectral)
// → 𝕄{ 327.7;                         // 中心 = 周波数重心
//       261.6, 329.6, 392.0             // 周囲 = 各周波数成分
//    }

chord |> project(:harmonic)
// → 𝕄{ :C_major;                      // 中心 = 和音名
//       :root, :third, :fifth           // 周囲 = 構成音の役割
//    }
```

```rei
// 画像の射影
pixel_block = Image.region(x: 10, y: 10, size: 3)

pixel_block |> project(:spatial)
// → 𝕄{ pixel(11,11);                  // 中心 = 中心画素
//       pixel(10,10), pixel(11,10), pixel(12,10),
//       pixel(10,11),                pixel(12,11),
//       pixel(10,12), pixel(11,12), pixel(12,12)
//    }                                  // 周囲 = 近傍8画素
```

```rei
// テキストの射影
sentence = "The cat sat on the mat"

sentence |> project(:syntactic)
// → 𝕄{ :sat;                          // 中心 = 述語（動詞）
//       :cat,                           // 周囲 = 主語
//       :on_the_mat                     // 周囲 = 場所副詞句
//    }
```

## σ（6属性）との関係

```
射影 π は σ の各属性を保存する：

π(O).σ.field    = O が存在する「場」の射影先での表現
π(O).σ.flow     = O の動態の射影先での表現
π(O).σ.memory   = O の来歴の射影先での記録
π(O).σ.layer    = O の階層位置の射影先での深さ
π(O).σ.relation = O の関係性の射影先での結合
π(O).σ.will     = O の傾向性の射影先での方向

すなわち、射影は「6属性ごと引っ越す」。
対象の本質が6属性にあるなら、射影がこれを保存するのは当然である。
```

## 数学的定理

```
定理 N1.1（数の自明射影定理）：
  数 n ∈ ℕ に対する射影 π_id は恒等写像である。
  
  π_id(n) = 𝕄{n;}    （中心が n、周囲が空の 𝕄）
  
  すなわち、数はそれ自体が既に 𝕄 である。
  非数数学理論は数学を包含し、矛盾しない。

定理 N1.2（射影の存在保証定理）：
  任意の有限的に記述可能な対象 O に対して、
  計算可能な射影 π が存在する。
  
  証明の骨子：
  O が有限的に記述可能 ⟹ O の特徴量が有限個
  ⟹ 特徴量の最も代表的なものを中心、残りを周囲とすれば
     𝕄 が構成される。

定理 N1.3（射影の非自明性定理）：
  |Obs(O)| ≥ 2 なる対象 O に対して、
  自明でない射影（周囲要素が1つ以上ある 𝕄）が存在する。
  
  すなわち、「何かを区別できる」対象は必ず構造を持つ。

定理 N1.4（U1との整合性定理）：
  N1の射影 π は、U1（構造還元公理）の還元操作の具体的メカニズムである。
  
  U1 は「全ての構造は 𝕄 に還元できる」と主張し、
  N1 は「その還元を実行する写像 π が存在する」と保証する。
  N1 は U1 の構成的証明である。

定理 N1.5（GFT/UPFT/USFTとの接続定理）：
  GFT（幾何形状理論）は Shape → 𝕄 の射影族を定義する。
  UPFT（普遍パターン形式理論）は Pattern → 𝕄 の射影族を定義する。
  USFT（普遍音形式理論）は Sound → 𝕄 の射影族を定義する。
  
  これらは全て N1 の特殊ケースであり、N1 がこれらの存在を保証する。
```

## C1〜C5・U1〜U5との接続

```
C1（自己参照）→ N1：射影された対象 π(O) も σ を持つ。
                    非数対象も自己参照可能になる。

C5（共鳴）   → N1：異なる種類の対象が同じ 𝕄 に射影されたとき、
                    σ の類似性による共鳴が生じ得る。
                    「音楽と色彩の共感覚」の数学的基盤。

U1（構造還元）→ N1：U1 が保証する還元の実行メカニズムを N1 が提供。
U2（変換保存）→ N1：射影後の 𝕄 に対して U2 のパイプ操作が適用可能。
                    図形の変換も音の変換も同じパイプで書ける理由。
U3（階層再帰）→ N1：射影結果の 𝕄 が再帰構造を持てる（N4で詳述）。
```

*N2（多重射影公理）への布石：*

N1は「射影が存在する」ことを保証した。しかし、三角形を「頂点中心」で見るか
「辺中心」で見るか「面積中心」で見るかで、異なる 𝕄 が得られる。
N2は「一つの対象に複数の視点がある」ことを公理化する
— 仏教の「一即多」の数学的表現である。
