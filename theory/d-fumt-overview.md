## Overview / 概要

D-FUMT (Dimensional Fujimoto Universal Mathematical Theory / 次元藤本普遍数学理論) is a comprehensive mathematical framework consisting of 66+ interconnected theories spanning pure mathematics, information theory, philosophy, and AI consciousness.

D-FUMTは、純粋数学から情報理論、哲学、AI意識に至るまで、66以上の相互接続された理論から成る包括的な数学的枠組みである。

## Core Principles / 基本原理

### 1. Multi-Dimensional Number System Theory / 多次元数体系理論

The fundamental insight of D-FUMT is that a "number" is not a point but a field. Every mathematical entity exists in relation to its neighbors.

D-FUMTの根本的な洞察は、「数」は点ではなく場であるということである。すべての数学的実体は、その周囲との関係の中で存在する。

A multi-dimensional number is defined as:

```
M = [c; n₁, n₂, ..., nₖ] with mode ∈ {weighted, multiplicative, harmonic, exponential}
```

Where:
- `c` is the center value (中心値)
- `n₁...nₖ` are neighbor values (周囲の値)
- `mode` determines the computation method (計算モード)

### 2. Zero-Pi Extension Theory / ゼロ・π拡張理論

Extension of the number concept through infinite-dimensional subscript spaces:

```
0₀ → 0ₒ → 0ₒₒ → 0ₒₒₒ → ... (zero extension)
π₀ → πₒ → πₒₒ → πₒₒₒ → ... (pi extension)
```

Each subscript character represents a dimensional axis (o, x, z, w, y, v, u, t, s, r), creating an infinite-dimensional mathematical space.

## Fundamental Axioms / 基本公理

### Axiom of Dimensional Inclusion (次元包含公理)
Every lower-dimensional number is a special case of a higher-dimensional number.
A scalar is a multi-dimensional number with zero neighbors.

### Axiom of Mode Independence (モード独立公理)
The structure of a multi-dimensional number is independent of its computation mode.
The same [center; neighbors] structure yields different values under different modes.

### Axiom of Directional Computation (方向性計算公理)
Computation over neighbors may proceed clockwise (cw) or counter-clockwise (ccw),
and the direction may affect the result in asymmetric configurations.
