## 問題提起

Reiの文書・コードにおいて「自己認識 (self-aware)」「自己参照 (self-referential)」「覚醒 (awakening)」といった用語が使用されている。これらは直感的に分かりやすいが、以下のリスクを含む：

1. **擬人化の危険** — 値が「意識」を持つかのような印象を与える
2. **数学的曖昧性** — 「自己認識」の厳密な定義が不明確
3. **学術的信頼性の毀損** — 査読者が「疑似科学」と判断する可能性

本文書は、これらの用語を**公理A3（σ蓄積）から導出される厳密な数学的概念**として再定義し、擬人化を排した表現に統一する。

|-----------------|----------------|-----------|

## 形式的定義

### 定義1: σ注釈付き値（旧: Self-Aware Value）

公理A3より、値空間V̂は生の値とσメタデータの対として定義される：

```
V̂ = V × Σ

ここで：
  v ∈ V  — 生の値（数値、MDim、拡張数など）
  σ ∈ Σ  — メタデータ（履歴、傾向性、変換回数）
```

**要点:** 「値が自分を知っている」とは、値vにσが付随していることの比喩的表現に過ぎない。σは値の性質ではなく、**値に外部から付与された注釈**である。ちょうど図書館の本に貸出履歴カードが付いているように、値にσが付いている。本が自分の貸出履歴を「知っている」わけではない。

### 定義2: σ豊穣度（旧: 覚醒度 / awareness）

σ豊穣度は、σに蓄積された情報の豊かさを[0, 1]のスカラーに正規化した指標である。

```
ρ(M, σ) = min(S / S_max, 1)

ここで S = s₁ + s₂ + s₃ + s₄ + s₅

  s₁ = min(|H| / 5, 1)          — 履歴の深さ
  s₂ = (τ ≠ rest) ? 1 : 0       — 傾向性の活性
  s₃ = min(n / 5, 1)            — 変換回数
  s₄ = min(|N(M)| / 8, 1)       — 構造の複雑さ（A1の周囲数）
  s₅ = |unique(H)| / |H|        — 履歴の多様性

  S_max = 5
```

**要点:** ρは純粋に数値的な指標であり、「意識」や「認識」とは無関係。「この値のσにどれだけ豊かな情報が蓄積されているか」を測定する関数に過ぎない。

### 定義3: σ飽和（旧: 覚醒）

σ飽和とは、σ豊穣度が飽和閾値θを超えた状態を指す。

```
σ-saturated(M, σ) ⟺ ρ(M, σ) ≥ θ    (θ = 0.6)
```

σ飽和した値は、evolve（T10）においてσの傾向性τに基づいたモード選択が行われる。これは値が「判断」しているのではなく、**蓄積された統計情報が選択関数の入力閾値を超えた**ことを意味する。

サーモスタットが設定温度を超えるとエアコンが起動するのと同じ原理である。サーモスタットは「暑い」と感じているわけではない。

### 定義4: σ傾向性（旧: 値の意志）

σ傾向性は、履歴Hの差分パターンから統計的に分類される方向ラベルである。

```
τ = classify(Δ)

ここで：
  Δᵢ = toNum(H[i+1]) - toNum(H[i])   for i = 1..|H|-1

  classify(Δ) = 
    expand    if 全て Δᵢ > 0
    contract  if 全て Δᵢ < 0
    spiral    if 符号が交互に反転
    rest      if 全て Δᵢ ≈ 0
```

**要点:** τは「意志」ではなく「統計的傾向のラベル」である。株価が上昇トレンドにあるとき、株価が「上がりたがっている」とは言わない。データパターンに名前を付けているだけである。

## 公理A3との関係

再定義後の概念体系は、全てA3から導出される：

```
A3: σ蓄積（全ての変換は痕跡を残す）
  │
  ├── σ注釈付き値 V̂ = (v, σ)     ← 旧: Self-Aware Value
  │     │
  │     ├── σ豊穣度 ρ(M, σ)      ← 旧: awareness（覚醒度）
  │     │     │
  │     │     └── σ飽和           ← 旧: awakening（覚醒）
  │     │           │
  │     │           └── evolveモード選択（T10）
  │     │
  │     ├── σ傾向性 τ            ← 旧: will（意志）
  │     │
  │     └── 6属性分解（T6）
  │           ├── field  = π_value(H)
  │           ├── flow   = π_diff(H)
  │           ├── memory = π_identity(H)
  │           ├── layer  = π_depth(H)
  │           ├── relation = π_refs(H)
  │           └── will   = τ（σ傾向性）
  │
  └── 不可逆性（σは蓄積のみ、消去不可）
```

**全てがA3の一つの原理「変換は痕跡を残す」から導出される。**  
擬人化は一切不要であり、あってはならない。

## 哲学的補足

「Self-Aware Value」という用語を排除することは、Reiの哲学的深さを損なうものではない。むしろ逆である。

仏教哲学における「空（śūnyatā）」は、あらゆる実体化・擬人化を否定する概念である。「値が自分を知っている」という表現は、値に固有の自性（svabhāva）を仮定しており、空の原理に反する。

σはvalue（値）の性質ではなく、**関係の網目の中で値に付随する条件的な注釈**である。値それ自体には何の自性もない。σが豊かなのは値が「賢い」からではなく、多くの変換関係を経たからである。これこそが縁起（pratītyasamutpāda）——全ては関係性の中で成り立つ——の数学的表現である。

したがって、擬人化を排した数学的再定義は、Reiの哲学的基盤をより正確に反映する。
