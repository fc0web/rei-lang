## Motivation / 動機

Modern mathematics, since Peano's axioms (1889), begins with the assumption that zero exists. Set theory (ZFC) begins with the empty set. But neither addresses the question: **how does zero itself come into being?**

現代数学は、ペアノの公理（1889年）以来、「ゼロが存在する」ことを前提としている。集合論（ZFC）は空集合から始まる。しかしいずれも、「ゼロそのものがどのように生じるのか」という問いには答えていない。

The Genesis Axiom System formalizes the emergence of number from pre-mathematical existence.

生成公理系は、数学以前の存在から数が生じる過程を公理化する。

## Phase Transitions / 相転移

```
void  →  ・(dot)  →  0₀  →  0  →  ℕ
      G-E₁        G-S₀     G-S₁    G-N₁
```

### Phase 0: Void (虚空)
Complete nothingness. Outside the axiom system. No structure, no value, no distinction.

完全な無。公理系の外部。構造なし、値なし、区別なし。

### Phase 1: ・(Dot / てん)
The first existence. A "something" without constraints.
Emergence from void is not a mathematical operation but an ontological event.

最初の存在。制約のない「何か」。虚空からの出現は数学的操作ではなく、存在論的事象。

### Phase 2: 0₀ (Zero-Zero / ゼロゼロ)
The birth of structure. Value and structure become distinct.
0₀ is not yet a number — it is the potential for number.

構造の誕生。値と構造が分離する。0₀はまだ数ではない——数の可能性である。

### Phase 3: 0 (Zero / ゼロ)
Value becomes fixed. Computation becomes possible.
This is where conventional mathematics begins.

値が確定する。計算が可能になる。従来の数学はここから始まる。

### Phase 4: ℕ (Natural Numbers / 自然数)
The number system emerges from zero. Peano arithmetic becomes applicable.

ゼロから自然数体系が生じる。ペアノ算術が適用可能になる。

## Axioms / 公理

### G-E₁: Axiom of Existence (存在公理)
Something can exist. We call it ・(dot).

何かが存在しうる。それを・（てん）と呼ぶ。

### G-S₀: Axiom of Structure Separation (構造分離公理)
From ・, 0₀ arises. Value and structure become separable.

・から0₀が生じる。値と構造が分離可能になる。

**Condition:** Curvature κ must exceed threshold κc × 0.5

### G-S₁: Axiom of Value Fixation (値固定公理)
From 0₀, 0 arises. Value becomes determined, and computation becomes possible.

0₀から0が生じる。値が確定し、計算が可能になる。

**Condition:** Curvature κ must exceed critical threshold κc

### G-N₁: Axiom of Number Genesis (数体系生成公理)
From 0, the natural number system ℕ emerges.

0から自然数体系ℕが生じる。

**Condition:** Structure index must exceed threshold 2.0

### FR: Firewall Rule (遮断規則)
・(dot) cannot be directly converted to a numerical value.
The transition must pass through 0₀. Phase transitions must advance exactly one step at a time.

・（てん）は直接数値に変換できない。遷移は0₀を経由しなければならない。相転移は一度に一段階ずつ進まなければならない。

## Philosophical Significance / 哲学的意義

The Genesis Axiom System bridges mathematics and ontology. It provides a formal framework for questions traditionally reserved for philosophy:

- **Creation ex nihilo:** How does something arise from nothing?
- **Emergence of structure:** When does unstructured existence gain form?
- **The nature of zero:** Is zero a number, a state, or a process?

In Buddhist philosophy, the concept of 空 (śūnyatā / emptiness) describes a state that is neither existence nor non-existence. The transition void → ・ in the Genesis Axiom System can be understood as a mathematical formalization of this concept.
