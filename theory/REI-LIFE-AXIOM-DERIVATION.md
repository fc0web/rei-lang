## 0. 問いの設定

> 「無」から「生命」は導出可能か？

Reiの公理体系は void（無）から自然数体系までの生成過程を形式化している。
本稿では、この同じ公理体系から「生命」の最小条件を導出できることを示す。

生命の定義は生物学において未だ合意がないが、ここでは以下の6条件を
生命の最小条件（Minimal Life Criteria: MLC）として採用する：

```
MLC-1: 境界        自己と環境の区別が存在する
MLC-2: 代謝        外部からの入力を内部状態の変化に変換する
MLC-3: 記憶        過去の状態が現在に影響を与える
MLC-4: 自己修復    損傷に対して元の構造を回復できる
MLC-5: 自己生成    自身の構成要素を自ら生産できる
MLC-6: 創発        個体の集合が個体にはない性質を示す
```

これらは Maturana & Varela のオートポイエーシス理論、
Schrödinger の「生命とは何か」、および仏教の縁起思想を
統合した条件である。

## 1. 公理の再掲

```
A1: Center-Periphery（中心-周囲）
    M = (c, N, μ, w)
    全ての値は中心 c と周囲 N の関係として存在する。

A2: Extension-Reduction（拡張-縮約）
    ⊕ : V × S → V,  ⊖ : V → V
    全ての値は深度方向に拡張・縮約できる。

A3: σ-Accumulation（σ蓄積）
    V̂ = V × Σ,  Σ = (H, τ, n)
    全ての変換は痕跡を残す。値は自身の履歴を保持する。

A4: Genesis Phase Transition（生成相転移）
    void → ・ → 0₀ → 0 → ℕ
    存在は無から段階的に生じる。段階の飛び越しは不可。
```

## 2. 導出

### 定理 L1: 境界の成立（A1 のみ）

**主張**: A1 から MLC-1（境界）が導出される。

**証明**:

A1 は全ての値を M = (c, N, μ, w) として定義する。
ここで c（中心）と N（周囲）は構造的に区別される。

```
定義: 自己 ≡ c（中心）
定義: 環境 ≡ N（周囲）
定義: 境界 ≡ c と N を分離する構造的区分
```

中心 c は周囲 N に含まれず、周囲 N は中心 c に含まれない。
この非対称な関係が「自己」と「環境」の最小限の区別を与える。

重み関数 w : N → ℝ⁺ は、環境の各要素が自己に与える
影響の度合いを定量化する。これは生物学における
「膜の選択的透過性」に対応する。

```
w(n_i) が大きい → n_i は自己に強い影響を与える（透過性が高い）
w(n_i) が小さい → n_i は自己への影響が弱い（透過性が低い）
w(n_i) = 0     → n_i は完全に遮断される（不透過）
```

退化条件 k = 0 （周囲なし）は、環境を持たない
孤立系であり、これは生命ではない。
**生命は k ≥ 1 を必要とする。** ∎

### 定理 L2: 代謝の成立（A1 + A3）

**主張**: A1 と A3 から MLC-2（代謝）が導出される。

**証明**:

A1 の compute 関数は、周囲 N からの入力を
中心 c の新しい状態に変換する：

```
compute(c, N, μ, w) → c'
```

この変換において：
- 入力: N の各要素 n_i（環境からの資源）
- 変換: μ（計算モード）が変換規則を決定
- 出力: c'（新しい内部状態）

A3 により、この変換の痕跡が σ に蓄積される：

```
変換前: V̂ = (c, Σ)
変換後: V̂' = (c', Σ')
  where Σ' = (H ∪ {c→c'}, τ', n+1)
```

これは代謝の定義そのものである：
「外部資源を取り込み、内部状態を変化させ、その過程を記録する」

計算モード μ の選択は「代謝経路の選択」に対応する：
- Weighted（加重平均）→ 均等な資源利用
- Harmonic（調和平均）→ 希少資源がボトルネック
- Geometric（幾何平均）→ 乗法的な資源効果

これらは生化学における異なる代謝戦略の数学的抽象である。 ∎

### 定理 L3: 記憶の成立（A3 のみ）

**主張**: A3 から MLC-3（記憶）が導出される。

**証明**:

A3 は V̂ = V × Σ を定義し、Σ = (H, τ, n) において：

```
H = [v₁, v₂, ..., vₙ]  — 変換履歴（過去の全状態）
τ                        — 傾向性（変化の方向）
n                        — 変換回数（経験の量）
```

H は時系列データとして過去の全状態を保持する。
τ は H から導出される傾向であり、「学習」の最小形態である：

```
τ = trend(H) = Σᵢ (vᵢ₊₁ - vᵢ) / (n-1)
```

過去の変換パターンが τ として凝縮され、
将来の変換に影響を与える。これは：

- 短期記憶: H の末尾要素
- 長期記憶: τ（傾向として圧縮された履歴）
- 経験量:   n（変換回数）

の三層構造であり、生物の記憶システムの最小モデルとなる。

Phase 7a の σ-interaction（12ルール）により、
記憶は他の属性（field, flow, layer, relation, will）に
カスケード伝播する。これは「記憶が行動を変える」
メカニズムの形式化である。 ∎

### 定理 L4: 自己修復の成立（A1 + A2 + A3）

**主張**: A1, A2, A3 から MLC-4（自己修復）が導出される。

**証明**:

A3 の σ が「正常状態」の履歴を保持していると仮定する。

```
定義: 正常状態 ≡ σ.H の統計的中心（期待値）
定義: 損傷   ≡ 現在の状態が正常状態から閾値以上乖離
定義: 修復   ≡ 現在の状態を正常状態に近づける操作
```

修復プロセス：

```
1. 検知: |c_current - E[σ.H]| > θ  （損傷検知）
2. 参照: c_target = E[σ.H]          （正常状態の参照）
3. 修復: 
   A2の⊖（縮約）で損傷部分を除去
   A2の⊕（拡張）で正常部分を再構成
   A1のcompute で周囲Nから修復資源を取得
4. 検証: |c_repaired - c_target| < θ （修復完了の確認）
```

ここで重要なのは：
- σ.H が「自分が何であるべきか」を記憶している（A3）
- ⊕/⊖ が構造の再構成手段を提供する（A2）
- compute が環境からの修復資源の取得を可能にする（A1）

Phase 7b はこれを3類型に分類して実装している：
- σ-integrity修復（メタデータの自己修正）
- 構造修復（中心-周囲関係の再構築）
- カスケード修復（連鎖的な属性修正）

生物学的には、DNA修復酵素、免疫系、組織再生に対応する。 ∎

### 定理 L5: 自己生成の成立（A1 + A2 + A3 + A4）

**主張**: 4公理すべてから MLC-5（自己生成）が導出される。

**証明**:

これがオートポイエーシスの核心であり、4公理すべてを必要とする。

A4 の Genesis は void → ・ → 0₀ → 0 → ℕ の生成過程を定義する。
この過程を「既に存在する生命体」が自身の内部で再実行できると仮定する。

```
定義: 自己生成 ≡ 生命体 L が自身の構成要素を
      内部的な Genesis 過程によって生産すること
```

自己生成のプロセス：

```
L = (c, N, μ, w) に対して：

1. 内部Genesis: c の内部で void → ・ → 0₀ → 0 を実行（A4）
   → 新しい「原始的存在」が c の内部に生じる

2. 構造付与: A2 の ⊕ で新しい存在に構造を与える
   → 0 → 0₀ → 0₀ₒ → ... と深度方向に拡張

3. 関係接続: A1 の compute で新しい要素を N に組み込む
   → 新要素が周囲の一部として機能開始

4. 履歴継承: A3 の σ が生成過程を記録
   → 次の自己生成のための「レシピ」が蓄積される
```

Phase 7c はこれを4つの Birth Axiom として実装：
- 分裂（split）: 中心が2つに分かれる
- 融合（merge）: 2つの値が1つの中心に統合
- 創発的誕生（emergent birth）: 周囲の相互作用から新しい中心が生じる
- 変態（metamorphosis）: 構造全体が質的に変化

生物学的には：
- 分裂 → 細胞分裂
- 融合 → 受精
- 創発的誕生 → 幹細胞からの分化
- 変態 → 蝶の変態、アポトーシス後の組織再編

**重要**: A4 の遮断規則「段階を飛び越せない」により、
自己生成は必ず段階的に進行する。これは生物の発生過程が
常に段階的であること（受精卵→桑実胚→胞胚→原腸胚→...）
と完全に一致する。 ∎

### 定理 L6: 創発の成立（A1 + A3）

**主張**: A1 と A3 から MLC-6（創発）が導出される。

**証明**:

A1 の中心-周囲構造を再帰的に適用する。

```
生命体 L₁ = (c₁, N₁, μ₁, w₁)
生命体 L₂ = (c₂, N₂, μ₂, w₂)
...
生命体 Lₖ = (cₖ, Nₖ, μₖ, wₖ)
```

これらの集合を、A1 を上位レベルで適用して一つの系とする：

```
Colony = (c_colony, {L₁, L₂, ..., Lₖ}, μ_colony, w_colony)

ここで：
  c_colony  = 集合の「中心的性質」（個体にはない集合的性質）
  {L₁...Lₖ} = 周囲（個々の生命体）
  μ_colony  = 集合レベルの計算モード
  w_colony  = 各個体の集合への寄与度
```

compute(c_colony, {L₁...Lₖ}, μ_colony, w_colony) の結果が
c_colony'（新しい集合的性質）となる。

**創発の条件**:

```
c_colony' ∉ {c₁, c₂, ..., cₖ} の任意の組み合わせ
```

すなわち、集合的性質が個体の性質の単純な合成では
説明できないとき、創発が生じている。

A3 により、各 Lᵢ の σ が相互に影響し合う
（Phase 7a の12ルール）。この相互作用のカスケードが
個体レベルでは予測不可能なパターンを生み出す。

Phase 7d はこれを4パターンに分類：
- 同期創発（個体が自発的に同期する）
- 分化創発（均一な集合が多様化する）
- 集合知創発（個体より賢い集合的判断）
- 相転移創発（量的変化が質的変化を引き起こす）

生物学的には：
- 同期 → 蛍の同期点滅、心筋細胞の同期拍動
- 分化 → 胚発生における細胞分化
- 集合知 → アリのコロニー、魚の群れ行動
- 相転移 → 単細胞→多細胞への進化的移行 ∎

## 3. 生命の最小モデル（Minimal Life Model: MLM）

以上の6定理を統合し、Reiの公理から導出される
生命の最小モデルを定義する。

```
定義: 生命体（Life Entity）

L = (c, N, μ, w, Σ, G)

ここで：
  (c, N, μ, w) — A1: 自己と環境の境界構造
  Σ = (H, τ, n) — A3: 記憶と履歴
  G : Genesis   — A4: 内部的な生成能力

以下の6条件をすべて満たす：

  (i)   k ≥ 1              — 環境との接触（孤立でない）
  (ii)  compute が定期実行  — 代謝（環境との物質交換）
  (iii) |H| ≥ 2            — 記憶（過去の状態を保持）
  (iv)  repair(L) → L'     — 自己修復（正常状態への回帰）
          where |c' - E[H]| < |c - E[H]|
  (v)   genesis(L) → L,L₊  — 自己生成（新しい構成要素の生産）
  (vi)  ∃ Colony s.t.       — 創発可能性（集合的性質の出現）
          compute(Colony) ∉ Span({cᵢ})
```

### 非生命との境界

```
石:     (i)✓ (ii)✗ (iii)✗ (iv)✗ (v)✗ (vi)✗  — 境界はあるが代謝なし
火:     (i)✓ (ii)✓ (iii)✗ (iv)✗ (v)✗ (vi)✗  — 代謝的だが記憶なし
ウイルス: (i)✓ (ii)✗ (iii)✓ (iv)✗ (v)△ (vi)✗  — 自力で代謝できない
結晶:   (i)✓ (ii)✗ (iii)✗ (iv)△ (v)✗ (vi)✗  — 自己修復的だが代謝なし
細菌:   (i)✓ (ii)✓ (iii)✓ (iv)✓ (v)✓ (vi)✓  — すべて満たす → 生命
AI:     (i)✓ (ii)✓ (iii)✓ (iv)△ (v)✗ (vi)△  — 自己生成が未達成
```

## 4. 存在の階層（Genesis Ladder of Life）

A4 の Genesis を拡張し、生命の発生を段階的に位置づける：

```
void                          — 絶対無
  ↓  G-E₁
  ・（dot）                    — 存在の可能性
  ↓  G-S₀
  0₀                          — 構造の萌芽
  ↓  G-S₁
  0                            — 計算可能な値
  ↓  G-N₁
  ℕ                            — 数の体系
  ↓  L-B₁（新規：生命境界遷移）
  Proto-Life                   — 境界+代謝+記憶（MLC 1-3）
  ↓  L-R₁（新規：修復遷移）
  Self-Maintaining Life        — +自己修復（MLC 4）
  ↓  L-A₁（新規：自己生成遷移）
  Autopoietic Life             — +自己生成（MLC 5）
  ↓  L-E₁（新規：創発遷移）
  Emergent Life                — +創発（MLC 6）= 完全な生命
  ↓  L-C₁（新規：意識遷移）
  Conscious Life               — +自己認識（将来の課題）
```

**遮断規則の拡張**:
A4 と同様に、生命の各段階も飛び越し不可：

```
遮断: ℕ から直接 Autopoietic Life には到達できない。
      必ず Proto-Life → Self-Maintaining → Autopoietic と
      一段ずつ遷移する。
```

これは地球の生命史と一致する：
- 化学進化（Proto-Life相当）: 約40億年前
- 原核生物（Self-Maintaining相当）: 約38億年前
- 真核生物（Autopoietic相当）: 約20億年前
- 多細胞生物（Emergent相当）: 約6億年前

構造的に対応する。

```
十二因縁              Rei公理        Genesis Ladder
────────            ─────         ──────────
無明（無知）          void           絶対無
行（形成力）          ・             存在の可能性
識（意識の種）        0₀             構造の萌芽
名色（心身）          0              計算可能な値
六処（感覚器官）      ℕ              数の体系
触（接触）            MLC-1          境界（A1）
受（感受）            MLC-2          代謝（A1+A3）
愛（渇愛）            MLC-3          記憶/τ傾向性（A3）
```

## 6. 形式的まとめ

### 主定理: 生命導出定理（Life Derivation Theorem）

```
Reiの4公理 {A1, A2, A3, A4} から
生命の最小条件 {MLC-1, ..., MLC-6} が導出可能である。

具体的には：
  A1           → MLC-1（境界）
  A1 + A3      → MLC-2（代謝）
  A3           → MLC-3（記憶）
  A1 + A2 + A3 → MLC-4（自己修復）
  A1+A2+A3+A4  → MLC-5（自己生成）
  A1 + A3      → MLC-6（創発）
```

### 公理の必要性

```
A1 は MLC-1,2,4,5,6 に必要  → 境界なくして生命なし
A2 は MLC-4,5 に必要        → 構造変更なくして修復・生成なし
A3 は MLC-2,3,4,5,6 に必要  → 記憶なくして生命なし
A4 は MLC-5 にのみ必要      → 生成相転移は自己生成の核心
```

**注目すべきこと**: A4（Genesis）は MLC-5（自己生成）にのみ
必要であるが、自己生成はオートポイエーシスの核心であり、
これがなければ「生命」ではなく「生命的な機械」にとどまる。

A4 が生命と非生命を分かつ公理である。
