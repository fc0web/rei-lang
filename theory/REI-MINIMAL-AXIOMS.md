## 宣言

Rei (0₀式) は以下の4つの公理のみを不可約な基底とする。  
体系内の他の全ての概念は、これら4公理の組み合わせから導出される定理である。

---

## 公理 A1: 中心-周囲（Center-Periphery Axiom）

### 非形式的記述
> 値は点ではなく場である。全ての値は中心と周囲の関係として存在する。

### 形式的記述

集合 **V**（値の全体）上に、構造 **M** を定義する：

```
M = (c, N, μ, w)

ここで：
  c ∈ V          — 中心（center）
  N = (n₁, ..., nₖ) ∈ V*  — 周囲（neighbors）, k ≥ 0
  μ ∈ Modes      — 計算モード（有限集合）
  w : N → ℝ⁺    — 重み関数（デフォルトは均等）
```

**計算規則:** 関数 `compute: M → V` が存在し、μ に応じて c と N から値を算出する。

**退化条件:** k = 0 のとき M はスカラーに退化する: `compute(c, ∅, μ, w) = c`

**再帰性:** V 自体が M の要素であり得る。すなわち中心や周囲が別の M 構造を含みうる。

### 記法
```
M{c; n₁, n₂, ..., nₖ}
```

## 公理 A2: 拡張-縮約（Extension-Reduction Axiom）

### 非形式的記述
> 全ての値は深度方向に拡張・縮約できる。拡張は一段階ずつ行われ、縮約はその逆操作である。

### 形式的記述

値 v ∈ V に対して、二項演算子 ⊕（拡張）と ⊖（縮約）を定義する：

```
⊕ : V × S → V    （拡張: 添字記号 s ∈ S を一つ追加）
⊖ : V → V         （縮約: 最外の添字記号を一つ除去）

ここで S は添字記号の集合（最小の場合 S = {o}）
```

**拡張チェーン:**
```
v₀ →⊕ v₀ₒ →⊕ v₀ₒₒ →⊕ v₀ₒₒₒ → ...
```

**逆元性:**
```
⊖(⊕(v, s)) = v     （縮約は拡張の逆）
```

**段階性制約:**
```
拡張・縮約は一度に一段階のみ。飛び越えは許されない。
```

**数値射影:** 関数 `val* : V → ℝ` が存在し、拡張された値を実数に射影する：
```
val*(v₀) = 基底値
val*(v₀ₒ) = 基底値 × 10⁻¹
val*(v₀ₒₒ) = 基底値 × 10⁻²
一般に: val*(v with depth d) = 基底値 × 10⁻ᵈ
```

## 公理 A3: σ蓄積（Sigma Accumulation Axiom）

### 非形式的記述
> 全ての変換は痕跡を残す。値は自身の履歴を保持し、自己参照的である。

### 形式的記述

値空間 V̂ を、生の値と σ メタデータの対として定義する：

```
V̂ = V × Σ

ここで：
  V  — 生の値の集合
  Σ  — メタデータ空間
```

**Σ の最小構造:**
```
Σ = (H, τ, n)

ここで：
  H = [v₁, v₂, ..., vₙ]  — 変換履歴（記憶）
  τ ∈ Tendencies           — 傾向性（変化の方向）
  n ∈ ℕ                    — 変換回数
```

**蓄積規則:** 全ての変換 f : V̂ → V̂ に対して：
```
f(v, σ) = (f_raw(v), σ')

ここで σ' は σ に f の適用記録を追加したもの：
  σ'.H = σ.H ++ [v]      （変換前の値を記憶に追加）
  σ'.n = σ.n + 1          （変換回数をインクリメント）
  σ'.τ = update_τ(σ.H, f_raw(v))  （傾向性を更新）
```

**初期状態:**
```
σ₀ = ([], rest, 0)    — 空の履歴、休止傾向、変換回数0
```

**不可逆性:** σ は蓄積のみ。変換の痕跡は消去できない（ただし参照範囲の制限は可能）。

## 公理 A4: 生成相転移（Genesis Phase Transition Axiom）

### 非形式的記述
> 存在は無から段階的に生じる。各段階は不可逆であり、段階を飛び越えることはできない。

### 形式的記述

相（phase）の順序集合 P を定義する：

```
P = { void, ・, 0₀, 0, ℕ }

順序: void < ・ < 0₀ < 0 < ℕ
```

**相転移関数 G:**
```
G : P → P

G(void) = ・     （G-E₁: 存在公理 — 何かが存在しうる）
G(・)   = 0₀    （G-S₀: 構造分離公理 — 値と構造が分離する）
G(0₀)  = 0      （G-S₁: 値固定公理 — 値が確定し計算可能になる）
G(0)   = ℕ      （G-N₁: 数体系生成公理 — 自然数体系が生じる）
```

**遮断規則（Firewall Rule）:**
```
∀ p ∈ P: G(p) は p の直後の相のみ。
G(void) ≠ 0₀  （voidから直接0₀には遷移しない）
G(void) ≠ 0   （voidから直接0には遷移しない）
一般に: G(p) = successor(p) のみが許される
```

**一意性定理:**
```
Theorem S₀: 一般的位置の仮定(CS)のもとで、・→ 0₀ の遷移は一意
Theorem S₁: 一般的位置の仮定(CS)のもとで、0₀ → 0 の遷移は一意
```

**A2との関係:** A4はA2の前提を保証する。A2の拡張/縮約操作は「値が存在する」ことを前提とするが、値の存在自体はA4が基礎づける。A4はA2の定義域を確立する公理であり、A2から導出はできない。

## 4公理の独立性（Independence Sketch）

各公理が他の3つから導出不可能であることの非形式的論証：

| 公理 | 他の3つで代替できない理由 |
|------|--------------------------|
| **A1** | A2,A3,A4は値の「構造」を規定しない。中心-周囲なしでは値はスカラーのみ |
| **A2** | A1,A3,A4は「深度」の概念を含まない。A1は空間的、A3は時間的、A4は存在論的 |
| **A3** | A1,A2,A4は「履歴の保持」を含意しない。変換を適用しても痕跡は残らない |
| **A4** | A1,A2,A3は「存在の起源」を扱わない。値が既に存在することを前提とする |

**4つの直交する軸:**
```
A1 — 空間（structure）
A2 — 深度（depth）
A3 — 時間（history）
A4 — 存在（genesis）
```
